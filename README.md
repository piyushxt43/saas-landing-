# Draftly - SaaS Landing Page

A modern, beautiful SaaS landing page built with Next.js, React, and Tailwind CSS.

## About

Draftly is a comprehensive SaaS platform designed to streamline business operations. This landing page showcases the platform's features, pricing, and benefits.

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **UI Components**: Radix UI
- **TypeScript**: Full type safety

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

To create a production build:

```bash
pnpm build
```
